# Hubscape ADK Agent Project Plan Template

This template is a standardized blueprint for planning, designing, and implementing custom agents under the Hubscape ADK (Agent Development Kit). It is designed to be **human-friendly** for team review and **AI-friendly** for parsing and implementation by coding agents.

> [!NOTE]
> If any Mermaid diagrams are used in the plan, authors **MUST** provide a backup clickable link directly below the diagram block to view/edit it in a web browser (e.g., using a [Mermaid Live Editor](https://mermaid.live/) share link), since many standard markdown readers do not natively render Mermaid syntax.
